### 如果出现安装不成功，可以通过以下方法再试试

1. 更新clover版本（EFI文件根目录下的**CLOVERX64.efi**文件）
2. 更新驱动（**CLOVER/kexts/Other**里的文件）
3. 如果还不行，那就只能找度娘了

## Catalina

#### bios版本为0077



### 5066EFI

#### bios版本为0073

#### bios版本为0073
